Pumpkin Juice Boilerplate
==========================

1 Gulp http://gulpjs.com/
2 Jade http://jade-lang.com/
3 ES6 
4 Stylus CSS Preprocessor http://learnboost.github.io/stylus/
  1 Jeet Grid System http://jeet.gs/
  2 Nib Prefixer and Mixins http://tj.github.io/nib/
  3 Rupture Media Queries http://jenius.github.io/rupture/


## How to use:

Clone this repo, then:
'''
npm install
'''

If you don't have gulp already, you'll need it:

'''
npm install gulp -g
'''

Then to build and start the watchers just run:

'''
gulp
'''
